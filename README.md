Toshimitsu Tsuji(tsujimitsu) personal web site.

## run

```shell
$ hugo server -D
```

## deploy

```shell
$ hugo -D
```

## create new article

```shell
$ hugo new posts/hello.md
$ hugo new posts/hello.md --editor="code"
```
