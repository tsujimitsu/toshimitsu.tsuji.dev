Electron Debugging

August 25, 2015

electron でデバッグを行う方法の１つ

  • electron-debug ツールを利用する

electron-debug の利用

    dir
    >index.html
     main.js
     ...

    npm install --save electron-debug
    dir
    >index.html
     main.js
     node_modules
       \electron-debug
     ...

    vim main.js
    >require('electron-debug')();

    vim package.json
    >"dependencies": {
       "electron-debug": "^0.2.0"
     }
    
    electron .
    >WindowsOSの場合は、F12でDev Tools起動、Ctrl+Rで再読み込み。
